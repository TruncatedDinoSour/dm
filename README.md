# DM - Download Manager

> A utility to help you download things...

## Dependencies
- python 3
- pip
- dbus (optional)
- a notification daemon (optional)
- linux

## Installation
```bash
$ make deps
$ make dm-full-setup
```

## Uninstallation
```bash
$ make uninstall
```

## Purging
```bash
$ make dm-purge
```

## Updating
```bash
$ dm-upgrade
```

## Updating (reset)
```bash
$ dm-upgrade full
```
